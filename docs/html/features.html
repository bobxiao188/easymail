内置特征列表

EasyMail 过滤引擎提供以下内置特征，可在过滤规则的条件表达式中直接引用。特征按功能分类组织，每个特征标注了数据类型、所属处理阶段和简要说明。

说明 下表中的"阶段"对应 SMTP 处理流水线：0=连接建立, 1=HELO, 2=发件人, 3=收件人, 4=邮件头, 5=正文。同一分类下的特征可能分布在多个阶段中。

一、连接信誉

在 SMTP 连接建立阶段采集，评估客户端的网络身份和信誉。

反向 DNS（rDNS）

特征键名类型阶段说明
ip_ptr_okbool0连接 IP 至少有一条 PTR 反向解析记录
ip_ptr_countnumber0PTR 记录数量
ip_ptr_has_dotbool0PTR 主机名中包含点号（完整域名检查）

正向确认反向 DNS（FCrDNS）

特征键名类型阶段说明
ip_fcrdns_okbool0FCrDNS 验证通过：PTR 名称的 A/AAAA 查询包含原始连接 IP
ip_ptr_forward_match_countnumber0前 3 条 PTR 中正向解析回原始 IP 的数量

IP 连接统计

特征键名类型阶段说明
ip_connect_count_1dnumber0该 IP 本日历日连接次数（Redis TTL 24h）
ip_connect_count_5mnumber0该 IP 滑动 5 分钟窗口连接次数
ip_connect_rate_1dnumber0本日连接频率（次数 / 自午夜秒数）
ip_connect_rate_5mnumber05 分钟连接频率（次数 / 300 秒）

IP 结果统计

特征键名类型阶段说明
ip_outcome_accept_1dnumber0该 IP 今日投递次数
ip_outcome_reject_1dnumber0该 IP 今日拒绝次数
ip_outcome_spam_1dnumber0该 IP 今日标记垃圾次数
ip_outcome_quarantine_1dnumber0该 IP 今日隔离次数
ip_outcome_total_1dnumber0该 IP 今日总处理次数
ip_reject_rate_1dnumber0该 IP 今日拒绝率
ip_reject_rate_5mnumber0该 IP 滑动 5 分钟拒绝率

二、HELO 检查

在 SMTP HELO/EHLO 阶段采集，验证客户端声明的身份标识。

特征键名类型阶段说明

特征在过滤规则中的使用

在管理后台的【过滤规则】页面，创建或编辑规则时，条件编辑器中的"选择特征"下拉列表包含了以上所有特征。

规则条件示例：

# 拒绝 SPF 验证失败的邮件
条件：spf_fail == true
动作：reject

# 对来自非本地联系人且包含短链接的邮件标记垃圾
条件：rcpt_contact_sender_hit == false AND body_url_has_short == true
动作：spam

# 对高频连接且无 PTR 的 IP 直接拒绝
条件：ip_ptr_ok == false AND ip_connect_count_5m > 50
动作：reject

# DMARC 失败且 DKIM 未通过时隔离
条件：dmarc_fail == true AND dkim_fail == true
动作：quarantine

规则按优先级顺序评估，匹配即终止（首条匹配规则生效）。
