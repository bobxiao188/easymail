Webmail 使用指南

EasyMail 提供基于浏览器的 Webmail 客户端。访问 http://服务器IP:8080，使用完整的邮箱地址和密码登录。

登录

在登录页面输入：

  • 邮箱地址：完整的邮箱地址，如 user@example.com
  • 密码：创建邮箱时设置的密码
提示 Webmail 和 IMAP 共享同一套用户凭据。登录 Webmail 后，IMAP 客户端也可以同时访问同一邮箱。

邮件管理

邮件列表

登录后默认进入收件箱，展示所有邮件列表。每封邮件显示：

  • 发件人
  • 主题
  • 日期和时间
  • 邮件大小
  • 是否包含附件
  • 阅读状态（已读/未读）
  • 标签

邮件操作

  • 阅读：点击邮件主题打开详情，查看发件人、收件人、正文和附件
  • 搜索：支持按发件人、收件人、主题、正文内容搜索邮件
  • 移动：选中邮件后移动到其他文件夹
  • 标记：标记为已读/未读、添加星标、设置标签
  • 删除：移入垃圾箱（可进一步永久删除或恢复）
  • 批量操作：支持多选后批量删除、移动、标记
  • 导出：下载邮件为 .eml 文件，或选中多封邮件打包为 ZIP 下载

写信

点击"写信"按钮，进入写信界面：

  • 收件人：直接输入邮箱地址，或从联系人列表选择
  • 抄送 / 密送：添加抄送和密送收件人
  • 主题：邮件主题
  • 正文：支持纯文本和 HTML 格式编辑
  • 附件：上传附件，支持拖拽
  • 保存草稿：未完成的邮件可以保存到草稿箱

回复与转发

  • 回复：回复发件人
  • 回复全部：回复所有收件人（包括抄送）
  • 转发：将邮件转发给其他人

文件夹管理

系统预设以下文件夹：

文件夹用途
收件箱接收的邮件
已发送已发送的邮件
草稿箱未完成保存的邮件
垃圾邮件被过滤规则判定为垃圾的邮件
垃圾箱已删除的邮件（可恢复）

用户可以创建自定义文件夹，将邮件分类存放。

联系人管理

联系人列表

在【联系人】页面可以查看和管理所有联系人：

  • 添加联系人（姓名、邮箱、电话、备注）
  • 编辑和删除联系人
  • 搜索联系人
  • 导入/导出联系人

联系人群组

  • 创建群组，将联系人分组管理
  • 批量发送邮件给群组成员

个人设置

账户设置

  • 修改密码
  • 修改个人资料（姓名、签名）

邮件设置

  • 每页显示邮件数
  • 回复时是否包含原文
  • 默认签名

安全退出

使用完毕后，点击右上角的退出按钮，安全退出 Webmail。
