快速入门

本文档引导完成从部署到成功收发邮件的完整流程。如果你已完成安装部署，可以直接从第一步开始。

业务流程图

管理后台(8000)                    Postfix 服务器
    │                                 │
    ├─ 1. 添加邮件域名 ───────────────┤
    ├─ 2. 创建邮箱账户 ───────────────┤
    ├─ 3. 配置 Postfix main.cf ──────┤
    └─ 4. 验证邮件收发 ───────────────┴─→ 5. 配置邮件客户端
1

添加邮件域名

登录管理后台 http://服务器IP:8000，进入【域名管理】页面。

点击"添加域名"，填写以下信息：

  • 域名：你的邮件域名，例如 example.com
  • 描述：可选的中文备注

添加成功后，域名列表中会显示新条目。此时域名状态为"启用"。你需要为该域名配置 MX 记录，将邮件指向你的 Postfix 服务器：

example.com.  IN  MX  10  mail.example.com.
mail.example.com.  IN  A   你的服务器公网IP
提示 如果只是内部测试，可以跳过 DNS 配置，直接在 Postfix 的 mynetworks 中放行内部发信 IP。
2

创建邮箱账户

进入【邮箱管理】页面（或在域名详情中点击"邮箱"选项卡）。

点击"添加邮箱"，填写：

  • 邮箱地址：例如 user@example.com
  • 密码：设置登录密码
  • 所属域名：选择上一步添加的域名
  • 邮箱容量：限制邮箱可使用的磁盘空间

创建成功后，该邮箱即可用于 IMAP 登录和收信。

注意 创建邮箱后，系统会自动在存储目录下创建该用户的文件夹结构（收件箱、已发送、垃圾邮件、垃圾箱等）。
3

配置 Postfix

在 Postfix 服务器上编辑 /etc/postfix/main.cf：

# 邮件域名列表（从 EasyMail 管理后台同步）
virtual_mailbox_domains = tcp:192.168.1.1:10025
virtual_mailbox_maps = tcp:192.168.1.1:10025

# LMTP 投递到 EasyMail
virtual_transport = lmtp:inet:192.168.1.1:10027

# Milter 反垃圾过滤
smtpd_milters = inet:192.168.1.1:10026
milter_default_action = accept

# SMTP 认证（通过 EasyMail 的 Dovecot 协议代理）
smtpd_sasl_type = dovecot
smtpd_sasl_path = inet:192.168.1.1:10025
smtpd_sasl_auth_enable = yes
smtpd_recipient_restrictions =
    permit_mynetworks,
    permit_sasl_authenticated,
    reject_unauth_destination
smtpd_sasl_security_options = noanonymous
local_recipient_maps =

将 192.168.1.1 替换为 EasyMail 服务器的实际 IP 地址。

修改后重启 Postfix：

systemctl restart postfix
注意 EasyMail 的 Dovecot 认证代理监听在 TCP 10025 端口，同时提供域名查询和认证服务。Postfix 通过 tcp: 协议查询域名是否存在，也通过 dovecot SASL 进行 SMTP 认证。
4

验证邮件收发

验证步骤如下：

4.1 确认 Postfix 能识别邮件域名

postmap -q example.com tcp:192.168.1.1:10025

应输出 OK 或邮件域名的相关信息。

4.2 发送测试邮件

从外部邮箱（如 Gmail、QQ 邮箱）向 user@example.com 发送一封测试邮件。或者使用命令行发送：

echo "Test email body" | mail -s "Test Subject" user@example.com

观察 EasyMail 日志，应该看到类似以下内容：

time="..." level=info msg="milter: filter result=accept" module=milter
time="..." level=info msg="lmtp: delivered mail to user@example.com" module=lmtp

4.3 检查邮件是否到达

通过管理后台的【投递日志】查看邮件投递记录。日志会显示每封邮件的发件人、收件人、过滤结果和投递状态。

4.4 使用 IMAP 客户端查看

使用任意 IMAP 客户端连接 EasyMail 服务器（端口 143），查看收件箱。配置参数：

  • IMAP 服务器：你的服务器 IP
  • IMAP 端口：143（明文）或 993（SSL）
  • SMTP 服务器：你的 Postfix 服务器 IP
  • SMTP 端口：587（STARTTLS）或 25
  • 用户名：完整的邮箱地址，如 user@example.com
  • 密码：创建邮箱时设置的密码

详细的客户端配置请参考 客户端配置指南。

5

配置反垃圾过滤

邮件系统正常收发后，建议配置反垃圾过滤策略。进入管理后台的【过滤规则】页面：

  • 查看默认过滤规则
  • 根据需要调整规则优先级
  • 配置分类模型（可选，需要预先训练）
  • 启用 ClamAV 病毒扫描（可选，需要安装 ClamAV）

详细配置请参考 后台功能指南 中的过滤章节。
