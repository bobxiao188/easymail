Email Client Configuration

EasyMail provides a full IMAP4rev1 service compatible with mainstream email clients. The following are the configuration parameters for each client.

Common Configuration Parameters

ParameterValue
IMAP ServerYour EasyMail server IP or domain
IMAP Port143 (plaintext) or 993 (SSL/TLS)
SMTP ServerYour Postfix server IP or domain
SMTP Port587 (STARTTLS) or 25 (plaintext)
UsernameFull email address, e.g. user@example.com
PasswordThe password set when creating the mailbox
AuthenticationNormal password (PLAIN / LOGIN)
Note Port 25 is often blocked by ISPs; it is recommended that clients use port 587 to send mail. If IMAP uses 993 (SSL), it is also recommended to use encrypted ports 465 or 587 for SMTP.

Foxmail Configuration

1

Open Foxmail, click "Tools" → "Account Management" → "New"

2

Enter your email address and password, then click "Manual Setup"

3

Fill in the server information

FieldValue
Incoming Server TypeIMAP
Incoming ServerYour EasyMail server IP
Incoming Port143
SSLUnchecked (or check if using port 993)
Outgoing ServerYour Postfix server IP
Outgoing Port587
Outgoing server requires authenticationChecked
4

Click "Test", confirm the connection is successful, then save

Outlook Configuration

1

Open Outlook, click "File" → "Add Account"

2

Select "Manual setup or additional server types", click "Next"

3

Select "POP or IMAP", click "Next"

4

Fill in the account information

FieldValue
Your NameDisplay name
Email Addressuser@example.com
Account TypeIMAP
Incoming Mail ServerEasyMail server IP
Outgoing Mail ServerPostfix server IP
User Nameuser@example.com (full email address)
PasswordMailbox password
5

Click "More Settings", switch to the "Outgoing Server" tab

  • Check "My outgoing server (SMTP) requires authentication"
  • Select "Use same settings as my incoming mail server"
6

Click "OK" to return, then click "Next" to test the configuration

Mobile Client Configuration (iOS / Android)

iOS Mail

  • Settings → Mail → Accounts → Add Account → Other
  • Add an IMAP account, enter your email address and password
  • Incoming Server: EasyMail IP, port 143, no SSL
  • Outgoing Server: Postfix IP, port 587, STARTTLS
  • Use full email address as the username

Android (Gmail App)

  • Open Gmail → Add account → Other
  • Enter your email address
  • Select IMAP (not Personal IMAP)
  • Enter your password
  • Incoming Server: EasyMail IP, port 143, security type "None"
  • Outgoing Server: Postfix IP, port 587, security type "STARTTLS"
  • Use full email address as the username

Verifying Configuration

After configuration:

  1. Try sending a test email to yourself
  2. Wait a few seconds and click "Receive"
  3. Check if the email appears in the inbox
  4. Reply to the test email to confirm the full send/receive loop works
Troubleshooting If the client configuration does not work, first check:
- Whether EasyMail and Postfix services are running properly
- Whether the mailbox account has been created and enabled
- Whether the firewall allows ports 143, 587, and 25
- Whether the full email address is used as the username
- Check EasyMail logs and Postfix logs for detailed error information
