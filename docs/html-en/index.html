EasyMail Enterprise Mail Gateway

Open-source, self-hosted mail security gateway. Provides anti-spam filtering, virus scanning, ML classification, and an admin panel in front of Postfix, with efficient delivery via LMTP.

Core Features

Admin Panel

Domain management, mailbox account management, filter rules, classification models, delivery logs — all operations in one web interface.

Anti-Spam Filtering

Milter protocol integration with Postfix, six-stage filter pipeline: connection analysis, HELO check, sender verification, recipient matching, header analysis, body deep inspection.

ML Classification Engine

Built-in FastText / DistilBERT classifier with ONNX Runtime inference support, customizable training samples and models.

Virus Scanning

Integrated ClamAV antivirus engine for real-time scanning of email attachments, with connection pooling and circuit breaker protection.

IMAP4rev1 Service

Full IMAP protocol support, compatible with Foxmail, Outlook, Thunderbird and other mainstream clients. Supports IDLE push for real-time email delivery.

Webmail

Send and receive emails from the browser, supports multi-tab management, contacts, attachment preview, and email search.

Service Monitoring

Heartbeat mechanism automatically detects service status, control panel visualizes mail traffic, top senders, and system health.

Quick Start

From deployment to first email delivery in about 30 minutes. Clear workflow guidance: domain → mailbox → Postfix → verification.

System Architecture

EasyMail does not handle SMTP inbound; it relies on Postfix as the edge MTA. The overall architecture is as follows:

External Mail ──→ Postfix (SMTP)
                │
                ├── Milter (10026) ──→ Anti-Spam Filter Pipeline
                │                        ├── Connection / HELO / Sender / Recipient Checks
                │                        ├── Header / Body Deep Analysis
                │                        ├── ML Classification Model Inference
                │                        └── ClamAV Virus Scanning
                │
                └── LMTP (10027) ──→ EasyMail Storage
                                        ├── MySQL (Management Data)
                                        ├── Redis (Cache + Statistics)
                                        ├── SQLite (Per-User Mail Index)
                                        └── Disk (Raw .eml Files)
                                        │
                                        ├── IMAP (143/993)    ← Client Mail Retrieval
                                        ├── Webmail (8080)    ← Browser Access
                                        └── Dovecot (10025)   ← Postfix SASL Auth
