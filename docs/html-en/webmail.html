Webmail User Guide

EasyMail provides a browser-based Webmail client. Visit http://ServerIP:8080 and log in with your full email address and password.

Login

On the login page, enter:

  • Email Address: Full email address, e.g. user@example.com
  • Password: The password set when creating the mailbox
Tip Webmail and IMAP share the same set of user credentials. After logging into Webmail, IMAP clients can also access the same mailbox simultaneously.

Mail Management

Mail List

Upon login, you will enter the inbox by default, showing the full list of emails. Each email displays:

  • Sender
  • Subject
  • Date and time
  • Email size
  • Whether it contains attachments
  • Read status (read/unread)
  • Labels

Mail Actions

  • Read: Click the email subject to open details and view sender, recipients, body, and attachments
  • Search: Search emails by sender, recipient, subject, or body content
  • Move: Select emails and move them to other folders
  • Mark: Mark as read/unread, add star, set labels
  • Delete: Move to trash (can be permanently deleted or restored)
  • Batch Operations: Multi-select to batch delete, move, or mark emails
  • Export: Download emails as .eml files, or select multiple emails and package them as a ZIP download

Compose

Click the "Compose" button to enter the compose window:

  • To: Enter an email address directly, or select from your contact list
  • Cc / Bcc: Add carbon copy and blind carbon copy recipients
  • Subject: Email subject
  • Body: Supports both plain text and HTML format editing
  • Attachments: Upload attachments, supports drag-and-drop
  • Save Draft: Unfinished emails can be saved to the drafts folder

Reply & Forward

  • Reply: Reply to the sender
  • Reply All: Reply to all recipients (including CC)
  • Forward: Forward the email to others

Folder Management

The system comes with the following preset folders:

FolderPurpose
InboxReceived emails
SentSent emails
DraftsUnfinished saved emails
SpamEmails flagged as spam by filter rules
TrashDeleted emails (recoverable)

Users can create custom folders to organize and store emails by category.

Contact Management

Contact List

On the Contacts page you can view and manage all contacts:

  • Add contacts (name, email, phone, notes)
  • Edit and delete contacts
  • Search contacts
  • Import / export contacts

Contact Groups

  • Create groups to organize contacts
  • Send bulk emails to group members

Personal Settings

Account Settings

  • Change password
  • Edit profile (name, signature)

Mail Settings

  • Number of emails displayed per page
  • Include original message when replying
  • Default signature

Sign Out

After use, click the logout button in the top right corner to securely sign out of Webmail.
